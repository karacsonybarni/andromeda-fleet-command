# Publish to GitHub

The repository is already initialized on branch main with one clean commit.

1. Create an empty public GitHub repository named andromeda-fleet-command.
2. Do not add a generated README, license, or .gitignore on GitHub.
3. From this project directory, run:

~~~bash
git remote add origin https://github.com/YOUR_USERNAME/andromeda-fleet-command.git
git push -u origin main
~~~

Then enable Issues and Discussions, add the topics godot, csharp, game,
open-source, steam, local-ai, voice-control, and space-strategy, and verify that
the CI workflow passes.
